# ClawCon Bogotá Landing Page

A minimal landing page for ClawCon Bogotá featuring a QR code that links directly to the organizer's LinkedIn profile.

## Prerequisites

- Node.js 18+
- npm 9+

## Setup

```bash
git clone git@github.com:Offthearc/clawcon-bogota-landing.git
cd clawcon-bogota-landing
npm install
```

## Run dev server

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Tests

```bash
npm test
```

## Lint

```bash
npm run lint
```
